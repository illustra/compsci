Lesson 8: Strings

Definition

It's simple: a string in C++ is an object that contains a group of characters.
We use strings in C++ to store names, error messages, etc.

Syntax

When using strings, we also need to include the strings library.
For example, this code:
				#include <iostream>
				#include <cstdlib>
				#include <string>
				using namespace std;
				
				int main()
				{
				    string name = "Aurel Jared Dantis";
				
				    cout << "Hello, my name is " << name << '!' << endl;
				
				    return 0;
				}
