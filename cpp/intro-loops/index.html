Introduction to Loops

Definition

Loops are control structures that execute given statements if the given
condition / boolean expression is true.

Elements

Repetition statement The type of loop used.
Test for limit condition / loop terminating condition The boolean expression that determines whether specified statement(s) will be executed.
Repeating code sections / loop body The statement(s) that will be executed if the boolean expression is true.
Loop index Keeps track of the number of times the loop body has been executed.
Update for loop index The statement responsible for changing the value of the loop index.

Types of Repetition Statements

while
do-while
for

Tips

To aid proper reading, use proper indents in your code.
Never make an infinite loop.
