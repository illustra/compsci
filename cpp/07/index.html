Lesson 7: Arrays

Definition

An array is a series of allocated memory spaces, one after the other (contiguous). Each
memory space is able to contain only the data type that the array was declared as.

Syntax

				// data_type array_name[numberOfMemorySpaces];
				short int gradeEight[8];

Modifying Array Elements

A programmer must declare each array element right after declaring the array itself, to
avoid garbage data being assigned to the array.

Now, to access an element in an array, we refer to the array's index. The index is
just a set of numbers beginning from 0 that refer to each element in the array.

For example, to access the first two elements in the array we declared above: 
				// array_name[indexNumber] = value;
				
				gradeEight[0] = 30;         // 30 students in Adelfa
				gradeEight[1] = 29;         // 29 students in Camia
				...

Full Example

				#include <iostream>
				#include <cstdlib>
				using namespace std;

				int main()
				{
				    gradeEight[0] = 30;      // 30 students in Adelfa
				    gradeEight[1] = 29;      // 29 students in Camia
				    gradeEight[2] = 30;      // 30 students in Champaca
				    gradeEight[3] = 29;      // 29 students in Dahlia
				    gradeEight[4] = 28;      // 28 students in Ilang-ilang
				    gradeEight[5] = 30;      // 30 students in Jasmin
				    gradeEight[6] = 29;      // 29 students in Rosal
				    gradeEight[7] = 30;      // 30 students in Sampaguita
				}
