Lesson 9: Two-dimensional Arrays

Definition

Like the one-dimensional arrays we discussed in Lesson 7, a two-dimensional array is a series of consecutive memory spaces that store values of the data type that the array was declared as.
The difference between 1D and 2D arrays is that 2D arrays have a concept of rows and columns, so each element's index contains its row and column number.
(Think of a 2D array as a Cartesian plane, and an array element is a point along that plane.)

Syntax

				// data_type arrayName[numberOfRows][numberOfColumns];
				int 2019periodicExamScores[8][30];

Modifying 2D Array Elements

To avoid illogical behavior in your program, it is best to give each element a value right after declaring the array.
To modify a single element, follow this syntax:

				// arrayName[row][column] = value;
				
				2019periodicExamScores[0][0] = 30;       // 1st student in Adelfa
				2019periodicExamScores[1][0] = 46;       // 1st student in Camia
				...

However, when managing large arrays, using loops are more efficient than hardcoding values for each element. See the example below:

				for ( int i = 0; i < numberOfRows; i++ )
				{
				    for ( int j = 0; j < numberOfColumns; j++ )
				    {
				        arrayName[i][j] = value;
				    }
				}
