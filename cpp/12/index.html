Lesson 12: Variable Scope

Definition

A variable's scope defines the area within the code where it can be used.

It can either be global (can be used anywhere in the code), or local (can be used only in the function where it was declared).


Example

				// Global variables are declared outside all function definitions
				int x;

				// Local variables can be used only in the function it was declared in
				int main()
				{
				    int y;
				}

Name Hiding and the Scope Resolution Operator (::)

If a local variable has the same name as a global variable, the local variable will be respected in the corresponding function; unless the scope resolution operator is used.

				// Global variable "x"
				int x = 1;

				int main()
				{
				    // Local variable "x"
				    int x = 2;

				    cout << x;     // The local variable will be displayed, e.g. "2"
				    cout << ::x;     // The global variable will be displayed, e.g. "1"
				}
