Lesson 2: cin, cout, endl, and Escape Sequence \n


Objects

What are objects? They're basically instances of code built from
a class. To understand it better, compare a class to a blueprint;
and an object is a house built from that blueprint.
We'll discuss more about classes (probably) in the coming weeks.

The two objects we discussed in class are 'cout' (console out)
and 'cin' (console in). The first one prints data on the screen,
the second one accepts inputs from the person using your program.

'cout' is followed by " << "data you want to display"; " while
'cin' is followed by " >> variableYouWantToStoreTheUserInputIn; " .
				#include <iostream>
				#include <cstdlib>
				using namespace std;
				
				int main()
				{
				    int variable1;
				
				    cout << "Enter a value:" << endl;
				    cin >> variable1;
				
				    cout << endl << "The value you entered was " << variable1 << endl;
				
				    return 0;
				}

endl and '\n'

These two basically do the equivalent of hitting the Enter
key on your keyboard; they move your cursor to the next line.
The only difference is where you put them.
				#include <iostream>
				#include <cstdlib>
				using namespace std;
				
				int main()
				{
				    cout << "endl goes at the end of the C++ statement" << endl;
				
				    cout << "while backslash-n goes inside quotes.\n";
				
				    return 0;
				}
