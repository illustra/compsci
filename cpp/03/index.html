Lesson 3: The if-else Statement

Purpose

It executes a statement (or a set of statements) if the condition
specified is true.
If the "else" keyword is specified, it will execute another statement
(or set of statements) if the condition is false.

Syntax

				    // if statement
				    if ( boolean expression )
				        // single statement
				    
				    if ( boolean expression ) {
				    {
				        // multiple statements
				    } 
				    
				    // if-else statement
				    if ( boolean expression )
				        // single statement
				    else
				        // single statement
				    
				    if ( boolean expression ) {
				    {
				        // multiple statements
				    }
				    else
				    {
				        // multiple statements
				    }
