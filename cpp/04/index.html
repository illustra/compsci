Lesson 4: Data Types in C++

Most of the time, we need different data types to store values in. Some types
are suitable for specific situations; others arent. It all really depends on what
you're going to use the value for.

Basic Data Types

short int 2 -32,768 ~ 32,767 - integer
int 4 (32-bit) -2,147,483,648 ~ 2,147,483,647 - integer

The const Modifier

This is a safety precaution programmers frequently use. When a variable is
declared as a constant variable, the value stored in that variable cannot
be changed anywhere on the program except on the declaration statement itself.
Constant variables are conventionally named with capital letters.

Example

				const float EVAT = 0.12;
				const short int PASSING_GRADE = 60;
