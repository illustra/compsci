Need to install Code::Blocks?
Code::Blocks 10.05 for Mac / Windows
Code::Blocks 13.12 for Mac / Windows

Using a Mac? You need to have the XCode Command-Line tools installed first before you can compile.
You can either download XCode from the Mac App Store (2.5 GB),
or download the gcc-only installer from here (280 MB, depending on OS X version).

Using Linux? Install g++ and g++-multilib using your distribution's package manager.

Need more help? Check the official documentation.
