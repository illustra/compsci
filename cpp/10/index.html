Lesson 10: Functions

Definition

Functions are independent blocks of code that perform a specific job or fulfill a specific purpose.
A function can call (execute) other functions, and can even call itself; but that is not recommended.
main() is a special function because it is always the first function that gets executed
when you run your program. That's why it's called the driver function, because it has the power
to call any other function when your program is run.


Syntax

				// Declare function first: data_type functionName(arguments);
				int squareRoot(int number);

				// Then define the function: data_type functionName(arguments) {}
				int squareRoot(int number)
				{
				    // statements go here
				}

Data Types and the return Statement

The data types you can declare a function as are essentially the same as the data types you can declare variables as.
The purpose of the data type is to determine what kind of value the function will "return"
to whatever function called it when it has finished running.

For example, if your function was declared as int, your return statement should contain an integer value or variable: 

				int function()
				{
				    return 0;
				}

				int anotherFunction()
				{
				    int someVariable = 1;
				    return someVariable;
				}

Same goes for pretty much every other data type, except for void, which does not return anything:

				void function()
				{
				    return;
				}

Basically, the return function does two things:
1) tell the system that the function has finished running, and
2) give back a value to whatever function called it, as a kind of "result".


Local and Global Variables

Local variables can only be used in the function where they were declared.
Global variables can be used in any function in the same program.

				// Declare global variables outside all functions
				float worldPopulation = 7.0e9;
				
				void Philippines()
				{
				    // Declare local variables inside the function they will be used in
				    float localPopulation = (worldPopulation / 7) / 10;

				    return;
				}

Arguments

Arguments are basically just variables that you give to a function so it has something to work on.

				int squareNum(int number);

				int main()
				{
				    int enteredValue;
				    
				    cout << "Enter a number: ";
				    cin >> enteredValue;
				    
				    cout << "The square of the number you entered is " << squareNum(enteredValue) << '.';
				    cout << endl << endl;
				    return 0;
				}

				int squareNum(int number)
				{
				    int square = number * number;
				    return square;
				}

In the example above, the main() function gives the number stored in enteredValue to
squareNum(), who then stores the value in the argument number.
It gets the square of number and stores the result in square, then gives the value
stored in square back to main() in order to be displayed.
