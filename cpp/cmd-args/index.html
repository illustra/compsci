Something More: The system() Function

Usage

This function is used to execute other programs in the system. Any console output that
is generated by the program called by system() is displayed on your C++ program's console screen.

Since system() calls programs natively or already present on the system it is being run on,
system() should never be used (or used very carefully) when releasing programs on different
operating systems.

Syntax

				#include <iostream>
				using namespace std;
				
				int main()
				{
				    // code here
				
				    system("program_name");
				}

Common Windows Program Names

Here are some examples of programs one usually calls with system() in non-production code.

Program Name Purpose
cls Clears the console screen
pause Waits for user to press any key before continuing
dir Lists all the files and folders in the current directory


Warning!

NEVER, EVER use system() for anything other than homework or playing with your own code. It is a very dangerous function and may elicit defensive responses from your antivirus, especially if your program is in a finalized state.
