Lesson 5: The while / do-while Loops


while

This loop repeats the given statement(s) while the boolean expression is true. The boolean expression
is evaluated before executing the loop body; thus it is called a precondition / pre-test loop.

Syntax

				    // Single statement
				    while ( boolean expression )
				        // statement
				    
				    // Multiple statements
				    while ( boolean expression )
				    {
				        // statement 1
				        // statement 2
				        // statement 3
				        // statement n
				    }

do-while Loop

This loop, unlike the while loop, does not check if the boolean expression is true
before it executes the statements for the first time; rather, it checks after the statement(s) is/are executed. Thus it is called a postcondition / post-test loop.

Syntax

				    do
				    {
				        // statement 1
				        // statement 2
				        // statement 3
				        // statement n
				    } while ( boolean expression );
