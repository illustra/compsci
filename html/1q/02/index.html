When making webpages, it is important that you follow a specific format, so that the browsers that you use to view your pages with will display the page correctly.

Setting the document type (DOCTYPE)

The document type tells the browser what kind of file it's viewing. In our case, since we're making HTML pages, we set the document type to HTML as follows:

				<!DOCTYPE html>
				<html>
					
				</html>

The html part will need to be changed as our content changes, but let's not focus on that right now.

Proper tag nesting

HTML tags can be nested, i.e. a tag can be placed inside another tag. Because of this, it is very important that you open and close your tags properly. See this example:

				<!-- This is improperly nested -->
				<table>
					<tr>
						<td>Hello world!</tr>
					</table>
				</tr>

				<!-- This is properly nested -->
				<table>
					<tr>
						<td>Hello world!</td>
					</tr>
				</table>

Generally, tags that were opened first should be closed last, tags opened next should be closed before the last tag, ad infinitum.

There are also tags that do not need closing tags, which we will discuss in the next lesson.
