We aren't confined to the boringness of Times New Roman in black when making webpages. We can make everything look the way we want to - using something called Cascading Stylesheets, or CSS for short.

Syntax

				selector {
					property: value1 value2 ... valueN;
				}

where selector can refer to an element, a .class, an #id, a ::pseudo-element. or a :pseudo-class. Go here to view all possible selectors.

Multiple selectors can have the same styles, too!

				selector1, selector2, ... selectorN {
					property: value;
				}

Just like HTML, you can also insert comments into a CSS file.

				/* I am a comment! */
				selector {
					property: value; /* I am another comment! */
				}
