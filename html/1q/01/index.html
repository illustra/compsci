HTML, which stands for Hypertext Markup Language, is the language used to make webpages like the one you're looking at now.

HTML files are no more than simple text files, so you would need a text editor to make your first webpage. Here are some recommendations:

  • Sublime Text (unlimited free trial, $70 license) — This is my personal favorite. Sublime Text is a highly customizable Python-based text editor that supports themes, plugins, and a myriad of syntaxes. Available for
  • Notepad++ (free) — This is a text editor that manages to be full-featured yet extremely simple to use. Available for
  • Brackets (free) — This is a text editor specifically designed for making webpages, equipped with the ability to extract styles from Photoshop documents and quickly edit CSS styles of an element just with a simple keystroke. Available for

Good old Notepad will work as well, if you can't be bothered to install a new text editor. You'll be missing out on features such as code completion and syntax highlighting, though. ;)
